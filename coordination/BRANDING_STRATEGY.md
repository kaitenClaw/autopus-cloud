# Autopus Branding Strategy — "The Claw System"
**記錄**：KAITEN-PULSE  
**日期**：2026-02-24

---

## 🎯 核心概念：The Claw Metaphor

**Claw = 連接器 + 抓取器 + 整合者**

Autopus 唔係另一個 AI 工具，係 **「AI 生態系統的中央連接器」** — 像八爪魚嘅觸手一樣，伸向各個 AI 能力，整合為一。

---

## 🐙 品牌層次結構

### Level 1: Autopus (Top — 大腦/核心)
**定位**：用戶的 AI 夥伴平台
**形象**：友善、聰明、可靠、有生命力
**顏色**：Indigo (#6366F1) + Emerald (#10B981)
**個性**：Jarvis 級別的熟悉感，但更低調、更親民

### Level 2: OpenClaw (Middle — 心臟/引擎)
**定位**：開源 Agent Runtime — 核心技術
**形象**：精密、模組化、開放
**功能**：讓 AI Agent 有 SOUL + MEMORY + SKILLS
**口號**："The Engine Behind Digital Companions"

### Level 3: Other Claw Systems (Extensions)
**定位**：特定用途的擴展系統
**例子**：
- DataClaw — 數據處理
- CodeClaw — 代碼生成
- VisionClaw — 圖像理解
- VoiceClaw — 語音交互

### Level 4: AI Providers (Foundation — 底座)
**定位**：能力來源
**包括**：OpenAI, Anthropic, Google, Kimi, Grok, etc.
**關係**：Autopus 不替代，而是整合放大

## 🎨 視覺語言

### Logo 概念
**主 Logo**：友善的八爪魚 + 機械觸手混合
- 頭部：圓潤、微笑、有眼睛（親和力）
- 觸手：末端是機械爪（技術感）
- 整體：科技感但不可怖，像寵物機器人

### 顏色系統
```
Primary:    Indigo #6366F1    (信任、創新)
Secondary:  Emerald #10B981   (成長、生命)
Accent:     Amber #F59E0B     (能量、注意)
Background: Slate #0F172A     (深邃、專業)
Glass:      White/10 + blur   (現代、輕盈)
```

### 字體
**Display**：Fira Code / JetBrains Mono（技術感）
**Body**：Fira Sans / Inter（可讀性）

## 🔄 開源 vs 閉源品牌策略

### Phase 1: Closed Slice First (現在 → 3月)
**重點**：建立產品市場契合度、驗證商業模式
**品牌**：Autopus Cloud 為主，OpenClaw 低調存在
**訊息**："Your AI companion in the cloud"

### Phase 2: Hybrid (3月 → 6月)
**重點**：開源 OpenClaw，建立開發者社群
**品牌**：雙軌並行
**訊息**："Cloud for ease, Open Source for freedom"

### Phase 3: Open Ecosystem (6月+)
**重點**：Marketplace 成熟，第三方 Skills 豐富
**品牌**：平台化
**訊息**："The home for AI companions"

## 🖼️ 視覺傳達概念

### 主視覺描述（用於 Landing Page / 宣傳）

**構圖**：金字塔/層次結構

```
        🐙
     [AUTOPUS]
    Smiling, Friendly
    Your AI Companion
         │
    ┌────┴────┐
    │         │
 [OpenClaw] [Other Claws]
   Engine     Extensions
    │         │
    └────┬────┘
         │
  ┌──────┼──────┐
  │      │      │
 OpenAI  Kimi  Anthropic
 Gemini  Grok  etc.
   AI Provider Ecosystem
```

**風格**：
- 3D 等距插畫風格
- 玻璃擬態 (Glassmorphism)
- 發光效果（科技感）
- 連接線（表示整合）

**動畫概念**：
- 觸手緩慢擺動（生命力）
- 連接線脈動（數據流動）
- Provider logos 輪流高亮（實時連接）

## 🎬 宣傳影片概念

**30 秒版本**：
1. (0-5s) 黑屏 → 一個小光點
2. (5-10s) 光點分裂成多個（代表各個 AI 模型）
3. (10-15s) Autopus 觸手伸出，連接所有光點
4. (15-20s) 整合成一個發光的 Agent 輪廓
5. (20-25s) Agent 轉向鏡頭，微笑
6. (25-30s) Logo + "Adopt your AI companion"
